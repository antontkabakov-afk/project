# Frontend

This workspace contains the React + Vite frontend for Crypto Tracker.

Primary docs live in the repo root [`README.md`](../README.md) and in [`../github-wiki/`](../github-wiki/). The frontend now uses `/api` by default, with a Vite dev proxy for local development and an Nginx reverse proxy in containers.

Common commands:

```powershell
npm ci
npm run dev
npm run build
```
